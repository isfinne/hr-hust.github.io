我的Windows效率软件

作为一个搞机小能手，怎么能没有各种效率软件，下面就介绍一下(在我体验的诸多所谓的效率软件里面)我个人觉得比较好用的吧。

1. 快捷软件启动器Rolan

习惯了桌面只有一个回收站的我，是怎么打开常用的软件的呢，最开始我是利用Windows的搜索功能，win+软件名称+Enter即可打开，虽然也很方便，但输入英文软件名称时Windows输入法的自动中文切换很是让人头疼，让我不禁怀念软件快捷启动的方式，于是我找到了Rolan。

如图，设置快捷键后(比如我是win+alt)，即可快速打开面板，点击自己添加图标即可打开，还可以进行分类(这个功能简直太棒了)。

详细使用可以看这篇干净利落的 Windows 快捷软件启动器：Rolan 2。

不想看教程的直接去官网下载即可。虽然软件很好用，但是很多功能付费才能使用(这种软件收费居然还是订阅式的49/年)。当然，对于我来说，快捷启动功能已经很好用了，就不吐槽了。

2. 文件搜索神器Everthing

everything是Windows下一款极其快速的文件搜索软件，搭配后面提到的quicker，特别好用！

由于网站在国外，下载可能很慢，细心的我已经为你准备好了安装包。

3. 鼠标神器Quicker

Quicker 的使用逻辑和软件启动器比较接近，在快捷调用的基础上增加了更深度的功能。它的亮点在于用户可以通过可视化编程的方式编写属于自己的动作，官方共享库中有意思的动作也是层出不穷，堪称 Windows 上的快捷指令。

5. 快速预览工具QuickLook

QuickLook 是一款免费的轻量级快速预览工具。它能够在不运行关联程序的情况下，通过敲击空格键快速预览文件内容。

这款免费小工具，让你的 Windows 也能用上「一指禅」：QuickLook | App+1

6. 快捷启动器&&实用工具箱Utools

万事皆插件，让效率倍增的免费定制工具箱——uTools丨App+1

总有一款插件适合你！

7. 微软的工具箱PowerToys

食用指南 虽然功能很多，但是总体上还不是很强大，可以当作玩具玩玩（叫做Toys的原因？？？）

开个玩笑，微软更新的速度还是很快的，期待它变得更好用！

小结

所谓的效率软件贵在精而不贵在多，能够加快你工作or学习的效率才是王道，适合的才是最好的。

不要为了使用软件而使用，让这个软件融入你的workflow才是提升效率的正确方式。

当然，要是你像我一样捣鼓各种稀奇古怪的软件就另说了
