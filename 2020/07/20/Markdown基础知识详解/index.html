Markdown基础知识(typora)

序. 编辑器

虽然说markdown用随便什么文本编辑器都能写，但一个好用（好看）的编辑器会让你写作时效率更高。我目前在用的编辑器是Typora，Typora 是一款支持实时预览的 Markdown 文本编辑器。简单，强大，这是Typora最大的特色。只专注于markdown让它显得无比的强大，无论是对原生markdown图片功能的改善，还是快捷键的使用都是为了提高写作的效率，而文档转换跟latex数学公式的支持让Typora的扩展性更强。最重要的是，Typora是免费的，去官网下载安装即可。Typora的详细使用请看参考教程 。

一. markdown基础语法（快捷键为Typora默认）

1.1 标题

  • 使用#表示标题，一级标题用1个#，二级标题用2个#，以此类推，最多六级，# 后需跟一个空格。

  • 快捷键Ctrl +1,2,3,4,5,6

1.2 字体

  • 斜体:
    1. *文字*
    2. _文字_
    3. Ctrl + I
  • 粗体：
    1. **文字**
    2. __文字__
    3. Ctrl + B
  • 斜粗体：
    1. ***文字***
    2. ___文字___
    3. Ctrl + B + I

1.3 各种线

  • 分割线 *** (+++或者---) 记得回车

  • 删除线 ~~ 文字~~

  • 下划线 <u>文字</u> 或者 Ctrl +U

  • 高亮 ==文字== line

  • 无序 1. 2.
